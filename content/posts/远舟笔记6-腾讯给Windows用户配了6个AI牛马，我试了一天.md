>远舟笔记·第6篇

你有没有想过：

有一天，你不用再自己找文件、清垃圾、查设置。

你只要说一句话：

> "帮我清一下C盘。"
> "看看最近有什么好看的电影。"
> "把不用的软件找出来。"

电脑自己就干了。

今天（5月21日），腾讯发布了操作系统级AI助手——马维斯（Marvis）。名字蹭的贾维斯，功能嘛……我第一时间装上试了，说说真实感受。

## 马维斯是什么

一句话：**它不是跟你聊天的AI，是替你干活的AI。**

DeepSeek、豆包这些，你问一个问题，它给你一段文字回答，剩下的你自己干。

马维斯不一样。你说一句话，它直接操作你的电脑——打开软件、清理文件、浏览网页、修改系统设置。它深入到了操作系统底层，把你的整台电脑变成了一个"可对话的对象"。

腾讯给马维斯配了6个AI Agent，组成一个小团队：

| AI角色 | 负责什么 |
|--------|----------|
| Marvis（主Agent） | 理解你的指令，拆解任务，分派给下面的Agent |
| Computer Agent | 电脑系统操作——查配置、改设置、修网络 |
| File Agent | 文件管理——搜索、整理、清理 |
| Browser Agent | 网页交互——打开网站、抓取信息 |
| App Agent | 应用管理——分析使用习惯、卸载清理 |
| Search Agent | 网络搜索——聚合公开信息 |

6个Agent，7×24小时在线，给你1000万免费Token。

听起来像画饼对吧？我逐个测了一遍。

## 我让6个AI牛马干了活

### Marvis（主Agent）：全程指挥

没记错的话，从始至终我没看到Marvis"亲自下场"干活——它的活就是理解我说的话，把任务拆解开，派给下面的Agent，最后把结果汇总汇报给我。

说白了，就是个部长。

下面的活，它分给了5个专员：

### Computer Agent：查电脑配置

我说了一句：「查看电脑配置」。

马维斯把活派给了Computer Agent，它直接读取了我电脑的硬件信息——CPU型号、内存大小、显卡型号，一条条列出来。

注意，**不是给我一段文字教我怎么查，而是直接帮我查了。** 以前我得右键"此电脑"→属性，或者在运行框里敲dxdiag，现在一句话搞定。

说实话，第一次看到它自己把硬件信息一条条列出来的时候，我的第一反应不是"好方便"，而是——"它真的知道自己在干什么。"

### File Agent：清理C盘

我说：「清理C盘垃圾文件」。

这次马维斯派了File Agent。它扫了一遍C盘，列出了可以清理的文件清单，然后弹出一个确认窗口——

**不是上来就删，而是先让我看清单，等我点了确认才执行。**

这个细节我很在意。一个能直接操作你电脑的AI，最怕的就是"自作主张"。马维斯在这方面做得很克制：能帮你干的它干，需要你拍板的它等你。

说真的，这种"边界感"在AI产品里挺少见的。

### Browser Agent：帮我查电影

我说：「用浏览器打开豆瓣，看看最近有什么好看的电影」。

Browser Agent打开了Chrome，导航到豆瓣电影页面，检索了正在热映和热门电影的评分信息，然后**自动关闭了浏览器**，最后马维斯给我汇报了结果：

> 正在热映中评分最高的是《给阿嬷的情书》（9.1）和《纵横四海》（8.8）；热门电影里《世界的主人》（9.1）、《挽救计划》（8.6）、《疯狂动物城2》（8.3）口碑突出，可以考虑优先观看。

它不只抓了数据，还帮你做了排序和小结。连"看什么电影"这种选择困难，它都想帮你解决了。

不过它顺手把浏览器关了这件事，我倒希望它能先问一声——毕竟万一我还想自己逛逛豆瓣呢。但说实话，那一瞬间我有点恍惚：它已经开始像"人类助手"一样收尾了。

### App Agent：分析不常用的软件

我说：「我有哪些app是长期不用的」。

App Agent扫描了我电脑上所有安装的应用，列出了清单，还做了分类：

- **重复功能的应用**：比如我同时装了MarkdownPad 2、Typora、Obsidian三个编辑器
- **一次性工具**：比如人行征信中心密码控件、Awesomium SDK
- **不再使用的开发工具**：比如Xshell 6、Fiddler、Postman

我说"卸载这几个"，它弹出了确认界面，列出了5个应用的图标和名称，然后帮我**打开了每个应用的卸载窗口**。

注意，是打开卸载窗口，让你自己点最后一步确认——省去了你在控制面板里到处找卸载入口的麻烦，但删除的决定权还是在你手里。

这种"干到一半把手伸过来让你接盘"的感觉，像极了靠谱的同事。

**补充一句**：马维斯左侧的「应用」列表里，也可以直接右键卸载。想偷懒就说话，想精准控制就自己点，两种方式都行。

### Search Agent：全网搜AI资讯

我说：「帮我找找最新的AI资讯」。

这次马维斯派了Search Agent。它没有打开浏览器，而是直接调用了内置的搜索能力，从全网抓取信息后整理成要点汇报给我——整个过程几秒钟就完成了，不需要像Browser Agent那样打开网页再一步步操作。

后来我试多了，发现一个规律：同样是"查信息"，马维斯会自己判断该派给谁。只是"搜信息"，走Search Agent；需要"操作网页"，才调Browser Agent。

有点像部门分工。

这个"部长"当得还挺称职。

## 办公室：看着AI替你干活

马维斯有一个很有意思的功能——「办公室」。

打开后是一个3D虚拟场景：Marvis戴着红围脖坐在主位，像极了一个部门主管。底下的工位上，File Agent和Browser Agent正在干活，旁边还有茶水区和跑步机。

右上角是Token消耗面板，限额1000万。图库和文档还在构建中，Token还在涨。不过按实际用量估算，1000万的额度还是很充裕的。

右侧是对话历史栏，所有指令、中间确认、执行结果一目了然，不用翻聊天记录就能找到上次的操作。

**第一次觉得"让AI干活"这件事有了画面感。**

到这里我开始意识到：腾讯想做的不是一个聊天机器人，而是一个"住在电脑里的AI员工"。

## 本地知识库：它把你的电脑当家

打开马维斯的第一眼，我发现一个细节：左侧导航不是"聊天记录"，而是「本地知识库」——应用、文档、图库、此电脑。

它把你的整台电脑当成了自己的家。

最让我惊喜的是**图库**功能。授权后，马维斯用本地模型扫描了我电脑上的1000多张照片，自动梳理成四大类：

- **人物印象**：人脸识别，按人物分组
- **足迹地点**：根据照片的GPS信息按城市分类，旅游出差党的福音
- **时光长廊**：按拍摄时间归类，春节、劳动节、端午节……每个节日一张时间线
- **图片识别**：根据图片内容语义分类

说实话，这个分类逻辑比很多专业相册软件都清晰。

不过也要坦白说：**1000多张照片的初次梳理比较慢**，而且很吃Token。我写到这篇文章的时候，图库和文档还在构建中，Token已经从最初的7万多涨到了29万。

这里要说明一下：马维斯提供两种模式，我全程用的是**效率模式**（端云协同，又快又准）。还有一种**隐私模式**，完全使用本地模型运行，文件0上传，断网也能用。隐私模式适合处理敏感数据的场景，但速度会慢一些。

文档功能目前还比较早期，搜索时需要给出比较精准的范围限定才能得到好结果，期待后续迭代。

不过话说回来，光是图库这一项，就已经够我回去翻一整个下午的老照片了。

「此电脑」功能相当于强化版的"我的电脑"，但它有一个让我眼前一亮的细节——浏览文件时保留了完整的路径层级，点进去之后返回非常方便，不用像Windows资源管理器那样来回倒腾。而且速度很快，文件多的文件夹也能秒开。

## 值不值得装

我作为一个天天和电脑打交道的文案写手，说说我真实的判断。

**适合装的人：**

- 电脑小白和长辈——一句话查配置、改设置、清垃圾，不用到处找入口
- 文件多到管理不过来的人——本地知识库+语义搜索，比Windows自带的搜索强很多
- 经常出差的人——手机远程操控电脑，"随身携带的个人电脑"（官方介绍的功能，iOS版预计6月中旬上线，我的手机是苹果的还没来得及测）
- 隐私敏感的人——隐私模式断网也能用，数据不上云

**要注意的：**

- **硬件门槛不低**：6核CPU + 16GB内存 + 固态硬盘，不是所有电脑都能跑
- 文档搜索精度还需提升
- 图库初次梳理较慢
- iOS版还没上线（预计6月中旬，等上线了我再补测手机远程操控电脑的体验）

**和WorkBuddy的区别**（我平时也在用）：WorkBuddy偏代码、文档生成、项目协作；马维斯偏系统操作、文件管理、跨端控制。不是一个赛道，更像互补。

## 写在最后

马维斯不是完美的——文档搜索还不够精准，图库梳理还需要时间，有些功能还在早期阶段。

但它让我看到了一个方向：**AI不只是用来聊天的，它可以真正住在你的电脑里，帮你干活。**

过去我们使用软件，以后可能会变成：

我们只负责提需求，AI负责操作电脑。

马维斯未必已经成熟，但它让我第一次真实感觉到——"AI操作系统"这件事，开始落地了。

就像钢铁侠对贾维斯说的那句经典台词——

> "I am confident that we can figure this out."

AI越来越像同事，  
我也还在学习怎么和它一起工作。

远舟，在AI时代摸索成长的一名文案写手。
